Gesture → Label mapping: Gesture 0 = Thumb + Index  |  Gesture 1 = Thumb + Middle  |  Gesture 2 = Thumb + Ring
How it works: EMG data streams continuously into a rolling cyclic buffer. When MediaPipe holds a gesture for Min confidence frames, it instantly snapshots the last Pre-trigger seconds from the buffer — capturing the muscle activation that led up to the gesture. A Cooldown period then blocks re-triggering.
